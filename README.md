# JS-Projects
###JavaScript Projects

Count Vowels- Counts the number of vowels in a string.

FizzBuzz- Logs the numbers 1-100. Logs Fizz if the numer is multiple of 3, Buzz if a number is a multipe of 5 and FizzBuzz if a number is a multiple of both 3 and 5.

PI to the Nth Degree- Calculates PI to the inputted degree.

Reverse a String- Reverses an inputted string and states if it is a palindrome. 

Reverse the Words- Similar to Reverse a String except the words are reversed instead of characters.

Dice Roller- Roll any amount of dice with any amount of sides.

Guess the Number- Guess a number between 1 and 10.

Fibonacci- Efficient memoized Fibonacci finder
